# Product Description

This project is an **Airbnb Data Analytics Dashboard** built using Streamlit.

The product allows users to explore Airbnb listing data and generate insights through interactive visualizations.

The dashboard is designed to resemble a **modern analytics dashboard similar to PowerBI**.

Users must register and login before accessing the dashboard.

---

# Target Users

- Data analysts
- Students learning data analysis
- Researchers exploring Airbnb data
- Anyone interested in short-term rental insights

---

# Core Features

Authentication System

- User registration
- User login
- User logout
- Session-based authentication

Data Exploration

- View raw Airbnb dataset
- Filter and explore data

Data Processing

- Handle missing values
- Clean dataset
- Prepare features for analysis

EDA Dashboard

- Price analysis
- Neighborhood analysis
- Room type distribution
- Review trends

Insights

- Summarize findings from data analysis

Chatbot Assistant

- AI assistant that explains dashboard insights

---

# Product Goals

The product demonstrates:

- data preprocessing
- exploratory data analysis
- dashboard development
- interactive visualization

# Success Criteria

The project is successful if users can:
